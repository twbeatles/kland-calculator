> 대한민국 No.1 부동산 종합 계산기 - 전월세, 매매, 대출, 세금까지 한 번에!

## ✨ 주요 기능

### 📋 전월세 계산기
| 기능 | 설명 |
|------|------|
| **비용 비교** | 전세 vs 월세 실질 비용 비교 분석 |
| **안전 진단** | 전세가율 기반 보증금 안전도 평가 |
| **반전세 전환** | 순수전세 ↔ 반전세 조건 계산 (NEW) |
| **중개보수** | 법정 중개수수료 자동 계산 |

### 🏢 매매 계산기
| 기능 | 설명 |
|------|------|
| **취득세/채권** | 부동산 취득 시 세금 및 국민주택채권 계산 |
| **전세 vs 매매** | 전세/매매 손익분기 분석 (NEW) |
| **내집가격 계산** | 월 상환 가능액 기준 구매 가능 주택가격 (NEW) |
| **임대수익률** | 수익형 부동산 투자수익률 분석 |
| **갭투자 ROI** | 전세끼고 매수 시 수익률 시뮬레이션 |
| **중개보수** | 매매 거래 중개수수료 계산 |

### 💰 대출 계산기
| 기능 | 설명 |
|------|------|
| **대출 계산** | 만기일시/원리금균등/원금균등 상환 시뮬레이션 |
| **상환방식 비교** | 상환방식별 총 이자 비교 차트 |
| **상환 스케줄표** | 12개월 원금/이자 상세 내역 (NEW) |
| **주담대 한도** | DSR/LTV 기준 대출 한도 역산 (NEW) |
| **주택연금** | 55세+ 월 연금 수령액 계산 (NEW) |
| **전세대출 한도** | 버팀목/카카오뱅크/시중은행 한도 비교 |
| **DSR/DTI/LTV** | 금융규제 비율 계산 |

### 🧾 세금 계산기
| 기능 | 설명 |
|------|------|
| **양도소득세** | 부동산 매도 시 예상 세금 (장기보유공제 포함) |
| **재산세/종부세** | 연간 보유세 계산 |
| **증여세** | 배우자/자녀/손자녀 증여세 계산 (NEW) |
| **상속세** | 상속재산 세금 계산 (NEW) |

### 🏆 청약 계산기
| 기능 | 설명 |
|------|------|
| **청약 점수** | 가점제 84점 만점 기준 점수 계산 |

### 🛠 도구 모음
- 평수 계산기, 전월세 전환, 보증보험료, 인상률 계산
- 등기비용, 문자 마법사, 이사 체크리스트, 용어 사전, 히스토리

## 🚀 시작하기

```bash
# 저장소 클론
git clone https://github.com/your-repo/kland-calculator.git
cd kland-calculator

# 브라우저에서 실행 (빌드 불필요)
start index.html  # Windows
open index.html   # macOS
```

> 📌 CDN 기반으로 동작하므로 별도의 npm install이 필요 없습니다.

## 🎨 주요 특징

### 💡 사용자 경험
- **다크 모드** 지원 (부드러운 전환 애니메이션)

- localStorage 캐싱 및 debounced 저장
- React.memo 적용
- 스크롤 시 동적 헤더 그림자

### 🔒 안정성
- ErrorBoundary 오류 폴백 UI
- 입력 유효성 검사 및 시각적 피드백
- Try-Catch localStorage 예외 처리
