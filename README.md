## ✨ 주요 기능

### 📋 전월세 계산기
| 기능 | 설명 |
|------|------|
| **비용 비교** | 전세 vs 월세 실질 비용 비교 분석 |
| **안전 진단** | 전세가율 기반 보증금 안전도 평가 |
| **중개보수** | 법정 중개수수료 자동 계산 |

### 🏢 매매 계산기
| 기능 | 설명 |
|------|------|
| **취득세/채권** | 부동산 취득 시 세금 및 국민주택채권 계산 |
| **생애최초 감면** | 12억 이하 주택 최대 200만원 감면 (NEW) |
| **임대수익률** | 수익형 부동산 투자수익률 분석 |
| **갭투자 ROI** | 전세끼고 매수 시 수익률 시뮬레이션 (NEW) |
| **중개보수** | 매매 거래 중개수수료 계산 |

### 💰 대출 계산기
| 기능 | 설명 |
|------|------|
| **대출 계산** | 만기일시/원리금균등/원금균등 상환 시뮬레이션 |
| **상환방식 비교** | 상환방식별 총 이자 비교 차트 (NEW) |
| **전세대출 한도** | 버팀목/카카오뱅크/시중은행 한도 비교 (NEW) |
| **DSR 계산** | 총부채원리금상환비율 계산 |
| **DTI 계산** | 총부채상환비율 계산 (NEW) |
| **LTV 계산** | 담보인정비율 계산 |

### 🧾 세금 계산기 (NEW)
| 기능 | 설명 |
|------|------|
| **양도소득세** | 부동산 매도 시 예상 세금 계산 (장기보유특별공제 포함) |
| **재산세/종부세** | 연간 보유세 계산 (1세대1주택 공제 지원) |

### 🏆 청약 계산기 (NEW)
| 기능 | 설명 |
|------|------|
| **청약 점수** | 가점제 84점 만점 기준 점수 계산 |

## 🚀 시작하기

### 실행 방법
```bash
# 저장소 클론
git clone https://github.com/your-repo/kland-calculator.git

# 폴더로 이동
cd kland-calculator

# 브라우저에서 실행 (별도 빌드 불필요)
open index.html  # macOS
start index.html # Windows
```

> 📌 **참고**: CDN 기반으로 동작하므로 별도의 npm install이나 빌드 과정이 필요하지 않습니다.

## 🎨 주요 특징

### 💡 사용자 경험
- **다크 모드** 지원 (시스템 설정 연동)
- **반응형 디자인** (모바일 최적화)
- **한글 금액 표시** (예: 3억 5천만원)
- **단위 토글** (원 ↔ 만원 빠른 전환)
- **실시간 계산** (입력 즉시 결과 표시)

### ⚡ 성능 최적화
- **localStorage 캐싱** - 중복 접근 최소화
- **Debounced 저장** - 입력 완료 후 저장
- **React.memo 적용** - 불필요한 리렌더링 방지

### 🔒 안정성
- **ErrorBoundary** - 오류 발생 시 안전한 폴백 UI
- **입력 유효성 검사** - 범위 검증 및 시각적 피드백
- **Try-Catch 보호** - localStorage 접근 예외 처리

## 🗂 프로젝트 구조

```
kland-calculator-main/
├── index.html          # 메인 애플리케이션 (SPA)
├── manifest.json       # PWA 매니페스트 (NEW)
├── sw.js               # Service Worker (NEW)
├── README.md           # 프로젝트 문서
└── .Jules/
    └── palette.md      # 디자인 팔레트 참조
```

### 📱 PWA 설치 방법 (NEW)
1. 모바일 브라우저로 index.html 접속
2. 주소창 오른쪽 "홈 화면에 추가" 버튼 클릭
3. 독립 앱처럼 사용 가능!
